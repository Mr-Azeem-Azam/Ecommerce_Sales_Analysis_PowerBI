## üé® Theme & Formatting
- **Background Color:** White (`#FFFFFF`)
- **Title Font Color:** Dark Blue (`#1F4E79`)
- **Card Background:** Light Grey (`#F3F3F3`)
- **Consistent Font:** Segoe UI / Calibri
- **Saved Theme:** `PowerBI_Theme.json` (applied across all visuals)

---

## üíæ How to Use
1. Download the `.pbix` and `.json` files.  
2. Open `.pbix` file in Power BI Desktop.  
3. Load the Excel data source (`Sales_Data.xlsx`).  
4. Apply theme:  
   `View ‚Üí Themes ‚Üí Browse for Themes ‚Üí Select PowerBI_Theme.json`.  
5. Interact with slicers to explore data insights.
